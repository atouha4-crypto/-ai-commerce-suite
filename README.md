# 🛒 AI Commerce Suite
> An AI-powered platform for product analysis, sentiment analysis, and automated marketing content generation

## ✨ Features
- 🔍 **Sentiment Analysis** — Analyze customer reviews in Arabic & English
- 💰 **Price Monitoring** — Track and compare competitor prices automatically
- ✍️ **AI Content Generation** — Generate product descriptions & ad copy
- 📊 **Interactive Dashboard** — Arabic-first analytics dashboard
